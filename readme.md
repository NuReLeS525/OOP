# Personal Account Management System

## Description
This project is a simple Python-based personal bank account management system.
It demonstrates object-oriented programming concepts such as classes,
encapsulation, operator overloading, and transaction tracking.

## Features
- Deposit and withdraw money
- Track transaction history
- Operator overloading (+ and -)
- UML class diagram
- Unit test cases

## Project Structure
- amount.py
- personal_account.py
- main.py
- test_account.py
- uml_diagram.png
- README.md

## How to Run
1. Make sure Python 3 is installed
2. Run the program:
```bash
python main.py
